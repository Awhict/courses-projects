如何开始使用 ArtBlock

只需几步，您就可以开始收藏、交易和创作自己的数字艺术品

1. 连接您的数字钱包

选择您喜欢的数字钱包（如 MetaMask），并连接到我们的平台。这将是您交易和存储数字藏品的安全方式。

2. 浏览和发现藏品

探索我们平台上丰富多样的数字艺术品，按照类别、价格或创作者进行筛选，找到您喜欢的藏品。

3. 购买或拍卖藏品

找到心仪的藏品后，您可以直接购买或参与拍卖。所有交易都在区块链上记录，确保安全透明。

4. 创建和发行您的藏品

如果您是创作者，可以轻松上传和发行自己的数字艺术品，设置版税并从后续销售中获得收益。

5. 分享和展示您的收藏

在社交媒体上分享您的数字藏品，展示您的收藏品味，并与志同道合的艺术爱好者交流。

6. 获得版税收益

作为创作者，您设置的版税将在每次转售时自动分配给您，确保您持续从作品中获得收益。
